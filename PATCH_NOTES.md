# Patch Notes

## v0.2.0 (2026-02-13)

### New Features
-   **File-based Note Storage**: Notes are now saved as text files in `data/My notes/` for better persistence and easy access.
-   **Save Success Notification**: Added a visual popup "Saved correctly!" when notes are saved.
-   **Direct PDF Export (Notes)**: Replaced browser print dialog with direct PDF file download for notes.
-   **Lesson PDF Download**: Added a "PDF" button to the lesson header to download the full lesson content as a paginated PDF.

### Improvements
-   **Persistence Fix**: Resolved issue where notes were not loading correctly after refreshing the page.
-   **UI Updates**: Added download buttons and notifications to `LessonPage`.

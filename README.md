# School AI Avatar

## Overview
An educational AI platform with an interactive avatar, specialized in AI and Cybersecurity.
Designed with a "Warm Professional" aesthetic and powered by RAG (Retrieval-Augmented Generation) for accuracy.

## Documentation
> **READ THIS FIRST**: [Thinking of contributing? Read the Architecture Doc](./ARCHITECTURE.md)

Detailed documentation on the tech stack, project structure, and RAG strategy can be found in `ARCHITECTURE.md`.

## Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
